OpenStride development has two independent paths: building and flashing firmware to real hardware,
or running radio simulations with BabbleSim. Both share the same environment and workspace setup.
ANT+ SDK access is optional (required for testing ANT+ transport) and stacks on top of either path.

## Build and flash to hardware

Complete these in order.
