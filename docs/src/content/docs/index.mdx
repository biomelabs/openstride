OpenStride is an open-source alternative to commercial running power meter devices.
It measures pace, cadence, power, ground contact time, vertical oscillation, and
leg spring stiffness, communicating with Garmin and other running watches via ANT+
and BLE — using hardware you can source yourself and firmware you can read and modify.
