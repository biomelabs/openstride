# openstride

Open-source foot pod providing running dynamics over ANT+ and BLE.

## What is openstride?

It is my attempt at making an open-source version of a [well-known running wearable](https://www.stryd.com/us/en) which shall not be named. It's mostly just a fun embedded engineering side project, though also a little bit out of spite for [unreasonable pricing models](https://www.stryd.com/us/en/store).

Obviously, if you want to use something on a daily basis or for actual training, you want something reliable and well-tested. This is neither (for now). But it sure was fun to make!

## Getting Started

[**Check out the docs**](https://biomelabs.github.io/openstride) to see how to set up your development environment, build and flash the firmware, and optionally enable ANT+ and BabbleSim support.
